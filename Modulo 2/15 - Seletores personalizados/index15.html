Tipos de seletores

Identificação "id"

Usa-se o seletor "id" para aplicar estilos à elementos únicos. Um exemplo disso, é a página do Capítulo 14 desse curso, onde foi utilizado uma Id no parágrafo que explicava a aplicação de fontes personalizadas. Logo, para aplicar a fonte estilizada naquele parágrafo único foi usado o seletor de identificação para diferenciar aquele parágrafo dos demais.

Quando usado um "id" no documento HTML, é preciso usar "#" para correspondê-lo em um arquivo CSS, ou em uma estilização local.

Classes "Class"

Usa-se o seletor "class" para aplicar estilos à vários elementos de uma vez. Uma aplicação possível para isso quando se quer estilizar todos os parágrafos de um subtítulo específico, assim, todas os elementos com a mesma classe serão estilizados com a mesma propriedade.

Quando usado uma "class" no documentos HTML, é preciso usar "." para correspondê-lo em um arquivo CSS, ou em uma estilização local.

Pseudo-class

São seletores que definem o estado de um elemento. O caso mais exemplar é a pseudo classe "hover", que define um estado do link. Aproveitando esse assuntos, podemos citar algumas pseudo-classes relacionadas ao link, tais como:

  • a:link = para deixar o link sempre em estado original
  • a:hover = para estilizar o link quando passar o mouse em cima
  • a:visited = para estilizar o link quando já clicado
  • a:active: = para estilizar o link quando ele estiver sendo clicado

Pseudo-element

São seletores que aplicam uma modificação em um elemento. Como é o exemplo da guia superior dessa página, que possui um simbolo de seta para esquerta no link de ir para o capítulo anterior e um simbolo de seta para a direita para ir para o link posterior.
