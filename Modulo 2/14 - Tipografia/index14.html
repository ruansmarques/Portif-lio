Tipografia

Anatomia das fontes

As letras possuem várias medidas em sua composição. A letra X é o ponto base para se definir a altura de todas as outras letras dentro de uma fonte, essa letra define a altura das letras minúsculas. Os componentes das alturas de uma fonte são:

  • Altura X
  • Altura das maiúscula
  • Ascendentes
  • Descendentes
  • Corpo

Cada letra também possui algumas características de estilização que as diferenciam das outras. Algumas dessas características são chamadas de componentes anatômicos de uma letra, alguns desses componentes são:

  • Serifas
  • Hastes
  • Filete
  • Arcos
  • Esporão
  • Vérticie
  • Terminal
  • Braço
  • Pernas
  • Pés
  • Espinha
  • Olho
  • Cauda

Um caso interessante é que a estilização das letras podem diminuir nosso esforço de leitura, visto que o cérebro humano identifica as palavras como um "bloco de letras", evitando identificar letra por letra para discernir a palavra. Um exemplo prático disso é o texto abaixo:

Tamanho das fontes

Existem váridas formas de configurar o tamnho das fontes em um arquivo Web, podendo ser definidas por:

  • Medidas absolutas (cm, mm, in, px, pt, pc)
  • Medidas relativas (em, ex, rem, vw, vh, %).

Essas alterações mudam o tamanho do corpo das letras. A escala mais recomenda é o uso do "px" e do "em", sendo que "16px" equivale à "1em".

Modelação de fontes

Para mudar a largura do corpo de uma fonte, utilizamos a tag "Weigth", que significa Peso.
Atenção para não confundi-la com as tags "Width" e "Height", que significam Largura e Altura.

Essa tag "Weigth" possui uma escala que vão de 100 até 900, mudando a largura da fonte entre esse intervalo.
Há também predefinições que algumas fontes possuem, podendo-se usar no lugar das escalas, as propriedades "lighter, normal, bold e bolder".

Também há a possibilidade de baixar fontes prontas da internet. Para isso, basta obter os arquivos ".otf" ou ".ttf" das fontes, logo basta usar o seletor "@font-face" no arquivo CSS e indexar os arquivos da font na url da font-face, em seguida tem que adicionar o formato "format" da fonte, sendo:

  • Para o formato ".ttf" usa-se:
    format('truetype')

  • Para o formato ".otf" usa-se:
    format('opentype')
