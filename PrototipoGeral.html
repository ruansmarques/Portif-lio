Tipos de listas

Lista ordenada

Basta usar a tag OL
Exemplo de Order List:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Começar a programar

Tipos de OL:

OL type="1"
Esta acima é uma lista de ordem numérica

  1. Exemplo 1
  2. Exemplo 2
  3. Exemplo 3
OL type="a"
Esta acima é uma lista de ordem alfabética
  1. Exemplo 1
  2. Exemplo 2
  3. Exemplo 3
OL type="i"
Esta acima é uma lista de numeração romana
  1. Exemplo 1
  2. Exemplo 2
  3. Exemplo 3
OL type="1" start="5"
Esta acima é uma lista de ordem numérica que começa pela ordem "5"
  1. Exemplo 1
  2. Exemplo 2
  3. Exemplo 3

Lista não ordenada

Basta usar a tag UL
Exemplo de Order List:

  • Acordar
  • Tomar café
  • escovar os dentes
  • Começar a programar

Tipos de UL:

OL type="disc"
Esta acima é uma lista do simbolo bolinha

  • Exemplo 1
  • Exemplo 2
  • Exemplo 3

OL type="circle"
Esta acima é uma lista do simbolo bolinha aberta
  • Exemplo 1
  • Exemplo 2
  • Exemplo 3

OL type="square"
Esta acima é uma lista do simbolo quadradinho
  1. Exemplo 1
  2. Exemplo 2
  3. Exemplo 3
